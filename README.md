# Iron Pulse Gym - Single Page Website

A sleek, responsive, and modern single-page website designed for a gym/fitness center. It showcases the gym's facilities, professional services, and membership plans using a clean user interface.

## Features

- **Fully Responsive:** Adapts perfectly to desktops, tablets, and mobile devices using CSS Grid and Flexbox.
- **Smooth Scrolling:** Seamless navigation when clicking on menu links.
- **Interactive Navigation:** Sticky navigation bar that changes appearance on scroll, along with a functional mobile hamburger menu.
- **Modern UI Design:** Includes a striking hero section, hover effects on cards, and highlighted pricing plans to drive conversions.
- **Iconography:** Integrated with Font Awesome for crisp, scalable vector icons.

## Technologies Used

- **HTML5:** Semantic structure and layout.
- **CSS3:** Custom styling, CSS Grid, Flexbox, variables, and animations.
- **JavaScript:** DOM manipulation for the mobile menu toggle and scroll events.
